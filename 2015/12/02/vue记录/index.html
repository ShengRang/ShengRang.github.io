    1. 热替换问题: webpack有一个开发用的服务器.
      热替换就是说你编辑了 .vue 文件之后网页立即自动刷新. 需要vue-hot-reload-api支持
      配置npm run dev 为webpack-dev-server —inline —progress —colors —hot —content-base build. 注意这里要有—inline, 热替换才能奏效.
